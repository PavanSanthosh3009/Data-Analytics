# HR Data Analytics

## 📌 Project Overview

This project focuses on analyzing HR data to uncover insights into employee demographics, workforce trends, attrition, salary distribution, and other key HR metrics.

## 🎯 Objectives

- Analyze employee demographics and workforce composition
- Identify employee attrition patterns
- Analyze salary and compensation trends
- Understand employee performance and satisfaction
- Create meaningful HR dashboards and visualizations
- Support data-driven HR decision-making

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Excel
- Power BI / Tableau

## 📊 Key Analysis

- Employee Attrition Analysis
- Department-wise Employee Distribution
- Gender and Age Demographics
- Salary Analysis
- Job Role Analysis
- Employee Satisfaction Analysis
- Experience and Performance Analysis

## 📈 Key Insights

The analysis helps identify patterns and trends in employee behavior and workforce data, providing insights that can help HR teams improve employee retention and make better workforce decisions.
